# Huffman_file_compression
A file compression and decompression tool using Huffman coding algorithm implemented in c/c++
⚠️ Note: This is a Command-Line Interface (CLI) tool. It must be executed via Terminal/CMD with the required arguments. Running it directly from the IDE will result in an "Arguments Error".

### Usage

**Open CMD** in the directory containing the compiled .exe & the file to compress or decompress

To compress a file:
huffman_tree.exe -c input.pdf output_file

To decompress a file:
huffman_tree.exe -d compressed_file.ece2103 restored.pdf
